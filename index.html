Desafío Divino - Aura Kingdom ES

Hola Enviados de Gea! Llegó un nuevo PVP a Aura Kingdom Español. Y bue... como me gusta programar se me ocurrió crear una pequeña herramienta para los que deseen tener a mano toda la info de este nuevo modo de juego.

Antes que pienses mal, no, no es un bot. Es un pequeño sitio web guía, a modo de aplicación, para entender la mecánica, conocer las recompensas, los items y como combinarlos para lograr mejores resultados dentro de la arena.

Esta app contiene:

  • Guia de la jugabilidad de la nueva arena
  • Lista de todos los items disponibles dentro del campo de batalla
  • Lista de las recompensas disponibles para intercambiar los Cristales de Espiritu Luminoso
  • Un visualizador interactivo para conocer como cada item puede ser combinado dentro de la arena 😀. Este visualizador es posible usarlo en pantalla junto con el juego. 🎉

Si no deseas instalar nada, puedes usar esta aplicación desde la web. Solo ten en cuenta que el visualizador dinámico no se podrá posicionar sobre el juego.

Version Web

Requisitos para instalar:

  • Solo 15Mb de espacio
  • Internet Explorer 9 o superior
  • Coneccion a internet. (Es posible que la primera vez que lo ejecutes aparezca una advertencia de Windows Firewall como la que se muestra debajo.)

¿Qué? ¿Internet Explorer?! Sí, leiste bien. Pero no te preocupes no se abrirá el navegador ni se convertirá en predeterminado, ni nada de ello. Solo se usará su 'motor de renderizado'

¿Porque IE? La verdad que cuando tenía gran parte del proyecto armado comencé las pruebas con la herramienta que lo convierte en aplicación de escritorio. El resultado fue una aplicación de 120Mb y un consumo de 300Mb aprox. de RAM sin estar haciendo nada! 😨 Inadmisible. 😓

Por lo que tuve que buscar alternativas que puedan dar un mejor rendimiento sin perder la idea. Me encontré con un proyecto bastante novato que permite el resultado deseado y con un mejor rendimiento: La aplicación pesa solo 15Mb y tiene un consumo de apenas 8~9Mb de RAM. ¿El sacrificio? Todo el código debía ser compatible con Internet Explorer 😅 Llevó un poco más de tiempo pero preferí el rendimiento al confort 😻
